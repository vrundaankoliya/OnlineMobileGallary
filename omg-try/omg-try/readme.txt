TWG Flash Uploader animations

This zip contains a flash animation for the TWG Flash Uploader.

progressbar_amin.swf: Copy this file to the img folder of tfu and replace the 
                      existing file. You can also use a different name and set 
                      it in the config. The files has to be in the img folder!
progressbar_amin.swf: The sources you can modify to your needs.

The animations are based on free animated gifs or free flashes I have found in the 
web. If you think this sources have any license restrictions please contact me.

If you have created your own set please send this back to me and I will add this
to the download page.

Have fun using TFU/JFU/WFU,
